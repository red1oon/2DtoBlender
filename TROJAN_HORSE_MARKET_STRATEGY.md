### Technical Risks

**Risk 1: Auto-classification accuracy drops with new building types**
- **Probability:** High
- **Impact:** Medium (frustrates users)
- **Mitigation:**
  - Machine learning from user corrections
  - Building type templates (airport, office, hospital)
  - Community-contributed patterns
  - Progressive disclosure (start with high-confidence only)

**Risk 2: Large DWG files (>100MB) crash the parser**
- **Probability:** Medium
- **Impact:** High (blocks enterprise deals)
- **Mitigation:**
  - Streaming parser (don't load entire file in memory)
  - Progress indicators (show it's working)
  - File size warnings (suggest splitting large projects)
  - Batch processing mode for Enterprise tier

**Risk 3: Dependency hell (PyQt5, ezdxf versions)**
- **Probability:** Medium
- **Impact:** Medium (installation friction)
- **Mitigation:**
  - Pinned dependency versions
  - One-click installer with bundled deps
  - Docker container option
  - AppImage for Linux
