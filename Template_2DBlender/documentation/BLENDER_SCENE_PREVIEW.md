## ‚öôÔ∏è BLENDER IMPORT PROCESS

### What Happens During Import:

1. **Load Geometry** from `Ifc_Object_Library.db`
   - Each object_type queries library for mesh data
   - Vertices, faces, normals loaded
   - Materials assigned from library

2. **Create Blender Objects**
   - For each JSON entry, create mesh object
   - Apply transforms (position, rotation)

3. **Position Objects**
   - Set location: `obj.location = position[X, Y, Z]`
   - World coordinates (not relative)

4. **Rotate Objects**
   - Set Z-axis rotation: `obj.rotation_euler.z = radians(orientation)`
   - 0¬∞ = facing +X (East)
   - 90¬∞ = facing +Y (North)
   - 180¬∞ = facing -X (West)
   - 270¬∞ = facing -Y (South)

5. **Organize Scene**
   - Group by room (collections)
   - Group by phase (collections)
   - Name objects from JSON

6. **Mark as Placed**
   - Update JSON: `"placed": true`
   - For tracking completion
